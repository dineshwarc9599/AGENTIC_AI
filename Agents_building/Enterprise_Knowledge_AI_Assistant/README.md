# Enterprise Knowledge AI Assistant

## 📌 Project Overview

The **Enterprise Knowledge AI Assistant** is an AI-powered intelligent assistant designed to help organizations efficiently retrieve templates, prior deliverables, and important project resources using natural language queries.

This system enables users to describe their requirements and receive context-aware recommendations, improving productivity, collaboration, and development efficiency.

---

## ⚠️ Note

The templates, prior deliverables, and supporting resources used in this project are **synthetically generated for demonstration and development purposes only**.

No confidential, proprietary, or real enterprise data has been used in this repository.

This project is intended to showcase the architecture, workflow, and capabilities of an AI-powered enterprise knowledge assistant system.

## ⚙️ How the System Works

1. User provides requirements or queries
2. AI assistant processes the request
3. Relevant templates and prior deliverables are retrieved
4. Semantic search identifies the most useful resources
5. Tailored recommendations are returned to the user

## ▶️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/Enterprise_Knowledge_AI_Assistant.git
```

### Navigate to Project

```bash
cd Enterprise_Knowledge_AI_Assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Backend

```bash
cd backend
python app.py
```

---

## ▶️ Run Frontend

```bash
cd frontend
streamlit run app.py
```
